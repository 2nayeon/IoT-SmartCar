# 🏎️ 아두이노 기반 자율주행 보조 시스템 (Smart-Car-System)

ABot과 아두이노를 활용하여 라인트래킹, 장애물 감지, 자동주차 및 원격 제어를 통합한 **자율주행 보조 시스템(ADAS)** 구현 프로젝트입니다. <br>
다양한 센서 데이터를 실시간으로 처리하여 차량이 주변 환경에 지능적으로 반응하도록 설계되었습니다.
<br>
<br>

## 👥 팀원 정보 및 프로젝트 기간
- **팀원:** 변가은, 이나연, 조혜인
- **프로젝트 기간:** 2025.11 ~ 2025.12 (약 6주)
<br>

## 1. 개발 목표 및 특징
- **자율주행 핵심 로직:** 라인트래킹 센서를 이용한 경로 유지 및 초음파 센서를 통한 장애물 회피.
- **지능형 주차 시스템:** 사용자의 블루투스 명령에 따른 자동 주차 및 자동 소환 기능 구현.
- **환경 적응형 제어:** 조도 센서를 활용하여 주변 밝기에 따라 전조등(LED)을 자동으로 점등.
- **실시간 원격 제어:** 블루투스 통신을 통해 스마트폰으로 차량의 움직임 및 모드 설정 가능.
<br>

## 2. 주요 기술 스택 및 하드웨어 구성

### 🛠 하드웨어 (H/W)
| 분류 | 부품명 | 주요 역할 |
| :--- | :--- | :--- |
| **Main** | Arduino UNO | 메인 컨트롤러 및 시스템 로직 제어 |
| **Chassis** | ABot (Smart Car Kit) | 차량의 본체 및 구동 프레임 |
| **Communication** | HC-06 (Bluetooth) | 스마트폰 앱과의 무선 통신 및 명령 수신 |
| **Sensor** | 초음파 센서 (Ultrasonic) | 전/후방 장애물 감지 및 거리 측정을 통한 주차 보조 |
| **Sensor** | 라인 트래킹 센서 | 하얀색 라인 도로 인식 및 경로 추적 |
| **Sensor** | 조도 센서 (Cds) | 주변 조도 감지 및 자동 전조등 제어 |
| **Actuator** | DC Motor & Driver | 차량의 주행 및 방향 전환 제어 |
| **Indicator** | LED & Buzzer | 자동 전조등 및 장애물 감지 시 경고음 출력 |

### 💻 소프트웨어 (S/W)
- **Language:** C/C++ (Arduino Sketch)
- **Tool:** Arduino IDE, Bluetooth Controller App
<br>

## 3. 주요 기능 및 로직
- **자율 주행 모드:** 라인트래킹 알고리즘을 통해 곡선 및 직선 도로를 이탈 없이 주행.
- **장애물 회피:** 주행 중 전방 장애물 발견 시 즉시 정지 및 경고 알림.
- **자동 주차/소환:** 초음파 센서로 가용 공간을 계산하여 정교한 'T자' 또는 '평행' 주차 로직 수행.
- **오토 라이트:** 조도 값에 따른 임계치 설정으로 야간 주행 시 시인성 확보.
